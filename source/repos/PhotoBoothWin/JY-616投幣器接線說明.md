# JY-616 投幣器接線與程式說明

本程式已支援 **JY-616** 多幣種投幣器。JY-616 為**脈衝輸出**型，需經 **Arduino**（或相容板）讀取脈衝後，由 Arduino 以串口傳給電腦。

## 硬體接線（JY-616 → Arduino）

| JY-616 線色 | 接至 Arduino | 說明 |
|-------------|--------------|------|
| 紅 (12V)    | 外部 12V 電源 | 投幣器電源，勿接 Arduino 5V |
| 黑 (GND)    | GND           | 與 Arduino 共地 |
| 白 (COIN)   | 數位腳（如 D2）| 硬幣脈衝輸出，每枚硬幣輸出對應脈衝數 |

- 投幣器可設定每種硬幣對應的脈衝數（例如 10 元 = 1 pulse、50 元 = 5 pulses）。
- 建議將投幣器上的脈衝速度開關設為**慢速**，較易穩定偵測。

## 程式端支援的格式（Arduino 需送出以下其一）

電腦端（PhotoBoothWin）在 **COM3、115200 baud** 接收文字，Arduino 每收到一次投幣就送**一行**（結尾 `\n` 或 `\r\n`）：

1. **PULSES=數字**（建議）  
   - 例：`PULSES=10` → 10 元、`PULSES=50` → 50 元  
   - 可設為 1 pulse = 1 元，或依你設定的脈衝對應換算成金額後送出。

2. **COIN=數字**  
   - 例：`COIN=50` → 50 元。

3. **純數字一行**  
   - 例：`10`、`50` → 直接視為金額（元）。

## Arduino 範例（僅供參考）

- 接線：JY-616 的 COIN（白線）接 Arduino D2，GND 共地。
- 每偵測到一個脈衝就累加（或依你設定的「幾脈衝 = 幾元」換算），投幣結束後送出一行到 Serial。
- 範例邏輯（1 pulse = 1 元）：

```cpp
// 設定：COIN 接 D2，Serial 115200，每枚硬幣送 PULSES=金額
const int COIN_PIN = 2;
int pulseCount = 0;
unsigned long lastPulse = 0;

void setup() {
  pinMode(COIN_PIN, INPUT_PULLUP);
  Serial.begin(115200);
  attachInterrupt(digitalPinToInterrupt(COIN_PIN), onPulse, FALLING);
}

void onPulse() {
  if (millis() - lastPulse > 50) {  // 防彈跳約 50ms
    pulseCount++;
    lastPulse = millis();
  }
}

void loop() {
  // 每 100ms 檢查：若脈衝已停止累加，代表一枚硬幣結束，送出金額
  static unsigned long lastSend = 0;
  if (millis() - lastSend > 100) {
    lastSend = millis();
    if (pulseCount > 0) {
      Serial.println("PULSES=" + String(pulseCount));
      pulseCount = 0;
    }
  }
}
```

- 若你的 JY-616 設定為「1 pulse = 10 元」，則在送出前將 `pulseCount` 乘上 10 再送，或改送 `PULSES=10`、`PULSES=50` 等對應金額。

## 電腦端設定

- 投幣器接在 **COM3**（Arduino 的 USB 序列埠）。
- 若 Arduino 顯示為其他 COM（如 COM4），請在裝置管理員中將該埠改為 COM3，或聯絡開發者修改程式中的 COM 埠設定。
- 確保沒有其他程式占用 COM3（例如關閉 RS232-ICT004.exe）。

完成以上接線與格式後，待機畫面應會顯示「剛剛收到：NT$ XX」。
