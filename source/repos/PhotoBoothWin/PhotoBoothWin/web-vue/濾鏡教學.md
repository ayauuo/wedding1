# 濾鏡教學

本文件說明「新增濾鏡」與「可調參數（CSS filter）」的位置與做法。

## 新增濾鏡的 4 個位置

1. **型別定義**
   - 檔案：`app/src/types/photobooth.ts`
   - 在 `FilterId` union 型別加入新的 `id`

2. **濾鏡選單**
   - 檔案：`app/src/components/photobooth/shoot/FilterOptions.vue`
   - 在 `FILTER_OPTIONS` 陣列加入 `{ id, label }`

3. **拍照頁預覽（大圖與縮圖）**
   - 檔案：`app/src/components/photobooth/shoot/ScreenShoot.vue`
   - 在兩個 CSS 區塊各加入對應 class：
     - 大圖：`.shoot-main-preview.filter--你的id`
     - 縮圖：`.thumb-frame__wrap .shoot-page.filter--你的id`

4. **合成輸出（Canvas filter）**
   - 檔案：`app/src/composables/usePhotobooth.ts`
   - 在 `getFilterCssForCanvas` 的 `map` 裡加入 `你的id` 對應的 CSS filter 字串

## CSS filter 可用參數

以下都可串接使用（由左到右依序作用）：

- `brightness(值)` 亮度，1 原始；>1 變亮；<1 變暗
- `contrast(值)` 對比，1 原始；>1 對比更高
- `saturate(值)` 飽和度，1 原始；>1 更鮮豔
- `hue-rotate(角度)` 色相旋轉（例如 `hue-rotate(20deg)`）
- `grayscale(值)` 灰階，0～1
- `sepia(值)` 復古棕，0～1
- `blur(像素)` 模糊（例如 `blur(2px)`）
- `invert(值)` 反相，0～1
- `opacity(值)` 透明度，0～1
- `drop-shadow(x y blur color)` 陰影（較少用於照片濾鏡）

### 範例

```css
.shoot-main-preview.filter--money {
  filter: brightness(1.1) contrast(1.15) saturate(1.2) hue-rotate(-5deg);
}
```
