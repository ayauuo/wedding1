# 新增濾鏡教學

本文件說明如何在專案中**新增一個濾鏡**。濾鏡會出現在「選擇濾鏡」畫面的列表中，使用者點選後可即時預覽，並在按下「下一步」時套用到合成輸出的照片上。

---

## 重要觀念

- **同一個濾鏡的效果**必須在 **4 個地方**都設定，且 **filter 字串要一致**，否則：
  - 畫面上預覽的效果會和最終輸出的照片不一樣。
- **濾鏡 ID** 請用英文、小寫、多字用連字號，例如：`sunset-orange`、`sakura-pink`。
- **顯示名稱** 在 `FilterOptions.vue` 的 `label` 設定，可自由寫中文，例如：`日落橙`、`櫻花粉`。

---

## 需要修改的 4 個檔案與順序

| 順序 | 檔案路徑 | 要做的事 |
|------|----------|----------|
| 1 | `src/types/photobooth.ts` | 在 `FilterId` 型別中新增一個 ID |
| 2 | `src/components/photobooth/shoot/FilterOptions.vue` | 在濾鏡列表中新增一筆選項（id + 顯示名稱） |
| 3 | `src/composables/usePhotobooth.ts` | 在 `getFilterCssForCanvas` 的 map 中新增「ID → CSS filter 字串」 |
| 4 | `src/components/photobooth/shoot/ScreenShoot.vue` | 在兩處 CSS 區塊各加一行「主預覽」與「縮圖」的 filter 樣式 |

下面用**同一個範例**貫穿：新增一個名為「晨光」的濾鏡，ID 為 `morning-glow`，效果為稍微偏暖、略亮。

---

## 步驟一：在型別中新增 FilterId

**檔案：** `src/types/photobooth.ts`

1. 開啟檔案，找到 `export type FilterId =` 底下的那一串 `'xxx' | 'yyy' | ...`。
2. 在**最後一個** `'matte'` 後面加上一筆新的 ID，格式為：`| '你的濾鏡-id'`。
3. 注意：每行結尾除了最後一筆外，都要有 `|`；最後一筆後面**不要**加 `|`。

### 修改前（節錄）

```ts
export type FilterId =
  | 'none'
  | 'slim-face'
  // ... 中間省略 ...
  | 'neon'
  | 'matte'
```

### 修改後（節錄）

```ts
export type FilterId =
  | 'none'
  | 'slim-face'
  // ... 中間省略 ...
  | 'neon'
  | 'matte'
  | 'morning-glow'
```

**儲存檔案。**

---

## 步驟二：在濾鏡列表中新增選項

**檔案：** `src/components/photobooth/shoot/FilterOptions.vue`

1. 開啟檔案，找到 `const FILTER_OPTIONS: { id: FilterId; label: string }[] = [`。
2. 在陣列**最後一筆**的 `},` 後面，新增一筆物件：`{ id: '你的濾鏡-id', label: '顯示名稱' },`。
3. 若這是最後一筆，物件後面用 `},` 或 `}` 皆可（陣列最後一項可以不加逗號，但加了也無妨）。

### 修改前（節錄）

```ts
const FILTER_OPTIONS: { id: FilterId; label: string }[] = [
  { id: 'slim-face', label: '瘦臉' },
  // ... 中間省略 ...
  { id: 'matte', label: '霧面' },
]
```

### 修改後（節錄）

```ts
const FILTER_OPTIONS: { id: FilterId; label: string }[] = [
  { id: 'slim-face', label: '瘦臉' },
  // ... 中間省略 ...
  { id: 'matte', label: '霧面' },
  { id: 'morning-glow', label: '晨光' },
]
```

**儲存檔案。** 此時畫面上會出現「晨光」按鈕，但點選後還不會有效果，因為尚未設定 filter 字串。

---

## 步驟三：在 usePhotobooth 中設定「合成輸出」用的 filter

**檔案：** `src/composables/usePhotobooth.ts`

1. 開啟檔案，找到 `function getFilterCssForCanvas`，底下的 `const map: Record<string, string> = { ... }`。
2. 在 map 的**最後一筆**（例如 `'matte': '...'`）後面加一個逗號，然後新增一行：  
   `'你的濾鏡-id': '你的 CSS filter 字串',`
3. **CSS filter 字串** 就是標準的 CSS `filter` 屬性值，例如：  
   `'sepia(0.15) brightness(1.05) saturate(1.1)'`  
   可組合多個函數，用空格分隔。常用函數見文末「常用 CSS filter 參考」。

### 修改前（節錄）

```ts
const map: Record<string, string> = {
  // ... 中間省略 ...
  'matte': 'brightness(1.04) saturate(0.85) contrast(0.92)',
}
return map[filterId] ?? 'none'
```

### 修改後（節錄）

```ts
const map: Record<string, string> = {
  // ... 中間省略 ...
  'matte': 'brightness(1.04) saturate(0.85) contrast(0.92)',
  'morning-glow': 'sepia(0.15) brightness(1.05) saturate(1.1)',
}
return map[filterId] ?? 'none'
```

**重要：** 這裡的字串請**複製下來**，步驟四的兩處 CSS 要貼上**完全相同的** filter 值，這樣預覽與輸出才會一致。

**儲存檔案。**

---

## 步驟四：在 ScreenShoot.vue 中設定「預覽」用的 filter（兩處）

**檔案：** `src/components/photobooth/shoot/ScreenShoot.vue`

同一個 filter 要在 **2 個 CSS 區塊**各加一行，效果才會一致：

- **主預覽圖**（大圖）
- **左側縮圖**（小圖）

兩行的 **filter 值必須與步驟三的 map 裡完全一致**。

---

### 4-1 主預覽圖的 filter

1. 在 `ScreenShoot.vue` 中搜尋：`.shoot-main-preview.filter--matte`
2. 會看到類似：  
   `.shoot-main-preview.filter--matte { filter: brightness(1.04) saturate(0.85) contrast(0.92); }`
3. 在這一行**下面**新增一行，把 `matte` 換成你的濾鏡 ID，`filter:` 後面貼上與步驟三**相同**的字串：

```scss
.shoot-main-preview.filter--morning-glow { filter: sepia(0.15) brightness(1.05) saturate(1.1); }
```

### 4-2 縮圖的 filter

1. 在同一個檔案中搜尋：`.thumb-frame__wrap .shoot-page.filter--matte`
2. 會看到類似：  
   `.thumb-frame__wrap .shoot-page.filter--matte { filter: brightness(1.04) saturate(0.85) contrast(0.92); }`
3. 在這一行**下面**新增一行，同樣把 class 名稱改成你的濾鏡 ID，`filter:` 與步驟三、4-1 **相同**：

```scss
.thumb-frame__wrap .shoot-page.filter--morning-glow { filter: sepia(0.15) brightness(1.05) saturate(1.1); }
```

**儲存檔案。**

---

## 完成後檢查

1. **型別**：若使用 TypeScript，存檔後不應出現「未認識的 FilterId」之類的錯誤。
2. **畫面**：進入拍照流程 → 拍完 → 選擇濾鏡，列表中應出現「晨光」；點選後主圖與左側縮圖應即時套用相同效果。
3. **輸出**：選「晨光」後按「下一步」，合成出來的照片應與預覽效果一致。

---

## 若只要「改現有濾鏡的效果」

不需要動型別或 `FilterOptions.vue`，只要改 **3 個地方**，且 **filter 字串保持一致**：

| 檔案 | 位置 |
|------|------|
| `src/composables/usePhotobooth.ts` | `getFilterCssForCanvas` 裡的 `map`，該濾鏡 ID 對應的那一行 |
| `src/components/photobooth/shoot/ScreenShoot.vue` | `.shoot-main-preview.filter--該id` 的 `filter:` 值 |
| `src/components/photobooth/shoot/ScreenShoot.vue` | `.thumb-frame__wrap .shoot-page.filter--該id` 的 `filter:` 值 |

三處改成同一個新的 filter 字串即可。

## 常用 CSS filter 參考

以下為常搭配使用的 CSS filter 函數（可多個用空格接在一起）：

| 函數 | 範例 | 說明 |
|------|------|------|
| `brightness()` | `brightness(1.1)` | 亮度，1 為原樣，>1 變亮 |
| `contrast()` | `contrast(1.05)` | 對比 |
| `saturate()` | `saturate(1.2)` | 飽和度，1 為原樣 |
| `grayscale()` | `grayscale(1)` | 灰階，1 為全灰 |
| `sepia()` | `sepia(0.4)` | 懷舊褐調，0~1 |
| `hue-rotate()` | `hue-rotate(90deg)` | 色相旋轉，單位 deg |
| `blur()` | `blur(2px)` | 模糊（預覽與合成都可用，但較耗效能） |

**範例組合：**

- 暖色、略亮：`sepia(0.2) brightness(1.05) saturate(1.1)`
- 冷色：`hue-rotate(190deg) saturate(1.1) brightness(1.03)`
- 黑白：`grayscale(1) contrast(1.1)`
- 復古：`sepia(0.4) contrast(1.08)`

## 快速對照表（新增一個濾鏡時）

| 步驟 | 檔案 | 新增／修改內容 |
|------|------|----------------|
| 1 | `src/types/photobooth.ts` | 在 `FilterId` 最後加 `\| '你的-id'` |
| 2 | `src/components/photobooth/shoot/FilterOptions.vue` | 在 `FILTER_OPTIONS` 陣列最後加 `{ id: '你的-id', label: '顯示名稱' },` |
| 3 | `src/composables/usePhotobooth.ts` | 在 `getFilterCssForCanvas` 的 map 裡加 `'你的-id': 'filter字串',` |
| 4a | `src/components/photobooth/shoot/ScreenShoot.vue` | 在 `.shoot-main-preview.filter--xxx` 區塊加一行 `.shoot-main-preview.filter--你的-id { filter: 同上字串; }` |
| 4b | 同上 | 在 `.thumb-frame__wrap .shoot-page.filter--xxx` 區塊加一行 `.thumb-frame__wrap .shoot-page.filter--你的-id { filter: 同上字串; }` |

照此教學操作，任何一位開發者都可以獨立完成新增或修改濾鏡。
