# 下載頁（相片／影片）

掃描拍貼機 QR code 後會開啟此頁，可切換「相片」與「影片」並下載電子檔。

## 使用方式

1. 將本資料夾整個上傳到您的伺服器（或靜態託管）。
2. 在拍貼機專案的 `.env` 設定 `VITE_DOWNLOAD_PAGE_BASE_URL` 為此頁的基底網址（不含查詢參數），例如：`https://your-server.com/download`。
3. 使用者掃描 QR code 後，網址會帶上 `?img=...&video=...`，此頁會讀取並顯示相片／影片與下載按鈕。

## 檔案

- `index.html`：頁面結構
- `style.css`：樣式（與拍貼機結果畫面風格一致）
- `script.js`：讀取網址參數、切換相片／影片、下載連結

本資料夾可單獨部署，不依賴 Vue 或建置流程。
